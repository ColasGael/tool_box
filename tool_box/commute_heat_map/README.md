# Configuration


## Commute time API

### Create Google API Key

For: the App

Follow the guide: https://developers.google.com/workspace/guides/get-started

To create a Google Cloud project with the following specificities:
- Enabled APIs: `Geocoding API` ; `Routes API`
- Enabled scope: N/A
- Credentials Type: `API Key`
- API restrictions: `Geocoding API` ; `Routes API`

NOTE: You need to enable `Billing` for these APIs, but they come with a free usage cap.
Reference: https://developers.google.com/maps/billing-and-pricing/pricing#routes-pricing

### Reference

- `Geocoding API` : https://developers.google.com/maps/documentation/geocoding
- `Routes API` : https://developers.google.com/maps/documentation/routes
